Python3 rewrites of the sample shell scripts.

disk_cpu_load.py
    Rewrite of samples/disk_cpu_load.sh. Tests CPU load during a bulk
    disk read using dd. Accepts the same --max-load, --xfer, and --verbose
    options as the original, plus an optional device argument. Reads
    /proc/stat directly for CPU time accounting. Requires blockdev and dd
    on PATH; no non-standard Python libraries needed.

disk_stats_test.py
    Rewrite of samples/disk_stats_test.sh. Verifies a disk is visible in
    /proc/partitions, /proc/diskstats, and /sys/block/, then confirms that
    I/O stats change after activity generated by hdparm. Requires hdparm
    on PATH; no non-standard Python libraries needed.

    Two bugs fixed from the original shell script:
      1. Line 56: `2&>` is not a valid bash redirect operator -- "2" was
         passed as a literal argument to hdparm and only stdout was
         redirected. Fixed with stderr=subprocess.PIPE.
      2. Lines 69-70: the /sys/block stat comparison was never performed;
         check_return_code was called with $? from the previous
         check_return_code call instead of a fresh comparison result.
         Fixed by explicitly comparing before/after values in Python.
